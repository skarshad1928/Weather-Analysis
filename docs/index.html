Weather Analysis

This project is a web-based Weather Analysis and Rain Probability Estimation System designed to collect and analyze real-time weather data for Charleston city, located in the state of South Carolina, United States of America. The application uses precise geographical coordinates (Latitude: 32.8546, Longitude: -79.9748) to fetch accurate and location-specific meteorological information. Through a Flask-based backend integrated with the OpenWeather API, the system retrieves essential weather parameters such as temperature, humidity, and atmospheric pressure for continuous monitoring and analysis.

Based on the collected weather conditions, the system applies rule-based logical conditions to estimate the probability of rainfall, which is particularly relevant for a coastal region like Charleston that experiences a humid subtropical climate. Higher levels of humidity and elevated temperatures are treated as key indicators for potential rainfall. The results are displayed through a user-friendly web interface, allowing users to interactively view live weather data and rainfall estimates.
